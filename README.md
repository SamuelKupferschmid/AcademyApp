# AcademyApp
Diese Applikation ist als Gruppenarbeit für das Modul oop2 (Objekt-orientiertes Programmieren 2) umgesetzt worden.
##Authoren
Michael Ineichen

Samuel Kupferschmid

##Features
- MVC umgesetzt
- Import von CSV
- Daten werden unter movies.dat gespeichert (Binär via Reflexion)
- Layout mit JavaFX
- CRUD Operationen implementiert

##Zusdatzfeatures
- Databinding zwischen Master (GridView) und Details (TextFields)
